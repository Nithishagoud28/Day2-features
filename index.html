JAVASCRIPT

Features on Js:

  • Light Weight & Interpreted : Javascript is lightweight and Interpreted in the browser without requiring compilation. This makes it fast and suitable for web development.
  • Asynchronous Programming : Javascript supports Asynchronous operations through callbacks,promises,and async/await.
  • Cross-Platform Compatibility: Javascript runs and multiple platforms (brower,node.js etc)making it highly versatile.
  • Event-Driven : Javascript excels at handling eventts like user interactions(clicks, keypresses.)making it ideal for interactive web applications.
  • Browser Compatibility: Modern Javascript runs seamlessly in all major browers, and polyfills or tranpilers like babel ensure backward Compatibility.
  • Rich Ecosystem and Libraries: Javascript has a vast Ecosystem including libraries and frameworks like react,angular,vue and tools like Node.js and express for backend development.
  • ES6(Modules): Javascript supports modular code with import and export.
  • Platform Independence : Javascript code can run on any platform with a browser or javascript engine(e.g Node.js)
  • Community and Escosystem : With millions of developers and open-source Libraries,Javascript has an extensive support community and tools like npm.

Defined and Undefined:

  • Defined: A variable is defined if it has been assigned an value, even if the value is numm or 0.

    • ex:let x=10: let y=null console.log(x,y)

  • Undefined: A variable is undefined if it has been declared but not assigned a value.

    • ex: let z;//defined but not assigned a value. console.log(z) output:undefined.
